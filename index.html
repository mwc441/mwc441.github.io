Chomp

Chomp is a digital board game which was written using Java and the Java Swing libraries. Chomp is a 2-player game which allows users to go against each-other, or against a bot, to select pieces on a board. Once a user/bot selects a piece on the board, every piece below and to the right of it will be removed. The upper-left corner piece is the forbidden piece and whoever chooses it loses!
